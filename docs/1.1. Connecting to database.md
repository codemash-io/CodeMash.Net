# CodeMash - Database as a Service - Connect to database

When you enable a database service in your [CodeMash dashboard page](https://cloud.codemash.io/db) we create a new MongoDb database for you. After that is done you are able to create collections using our provided tools in the dashboard. After you create a collection you can start adding records through the dashboard or using provided .NET SDK.

Our database SDK can be downloaded as a Nuget package **CodeMash.Repository**.

To make API calls using .NET SDK firstly you need to register your repository. You can do that in a few ways.

1. Calling **_CodeMashRepositoryFactory_** method **_Create_** and passing your **API key** and **Project ID**

```csharp
 
using System;
using CodeMash.Repository;

namespace MyApplication
{
    /* Your custom data type */
    [CollectionName("Actors")]
    public class Person : Entity
    {
        public string Name { get; set; }
            
        public string Email { get; set; }
    }
    
    class Program
    {
        static void Main(string[] args)
        {
            var projectId = Guid.Parse("{YOUR_PROJECT_ID}");
            var apiKey = "{YOUR_API_KEY}";
            
            var personRepository = CodeMashRepositoryFactory.Create<Person>(apiKey, projectId);
        }
    }
}
```

2. Calling **_CodeMashRepositoryFactory_** method **_Create_** and passing your **settings file**. Settings file needs to have your **API key** as **"CodeMash:ApiKey"** and your **Project ID** as **"CodeMash:ProjectId"**.

```csharp
 
using System;
using CodeMash.Repository;

namespace MyApplication
{
    /* Your custom data type */
    [CollectionName("Actors")]
    public class Person : Entity
    {
        public string Name { get; set; }
            
        public string Email { get; set; }
    }
    
    class Program
    {
        static void Main(string[] args)
        {
            var personRepository = CodeMashRepositoryFactory.Create<Person>("appsettings.json");
        }
    }
}
```
